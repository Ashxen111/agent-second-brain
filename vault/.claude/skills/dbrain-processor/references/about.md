## Anti-Patterns (AVOID)

When processing entries DO NOT:

- Создавать абстрактные заметки без конкретных действий
- Делать задачи расплывчатыми типа "подумать о...", "изучить..."
- Дублировать контент без синтеза и выводов
- Создавать списки без приоритетов и дедлайнов
- Добавлять задачи, которые не связаны с финансами или обучением (Decision Filters)
